#### 변수 이름의 예

* 첫 부분에 숫자 쓰면 안됨
* 사실 별 규칙 없음. 그냥 특수문자만 쓰지 말라고 함.

input 함수 : 내가 입력한 값이 다시 함수 뒤로 넘어가서 적용이 됨. 입력한 값을 **문자열**로 반환한다.

##### type
* 정수 : int
* 부동 소숫점: float
* 문자열: str
* boonlean: bool

이해가 안 될때는 이게 함수라는 점을 생각해보자.
<p>
y=f(x)
y=f(f(x))
</p>
이런식으로 하나하나 펴가면 됨

ex:
```python
digit1 = int(input())
digit2 = int(input())

print(digit1 + digit2)
```

**소수점 빼기**
```python  
interest = 0.087  
print (format(interest,".2f"))
```

2f: 두 번째 자리까지 표시하라는 뜻

**입출력 관련 예제**
```python  
num1=int(input())
num2=int(input())
print(num1+num2)
```
사용자에게 두 개의 함수를 입력받고 그 값을 더하는 식
틀린 이유: 정수로 변환했어야 함
